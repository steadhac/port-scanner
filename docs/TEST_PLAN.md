# Test Plan

## Objective
Verify the correctness, reliability, and robustness of the port scanner.

## Scope
- Command-line argument parsing
- TCP port scanning (OPEN, CLOSED, FILTERED)
- HTTPS certificate retrieval
- Error handling

## Test Types
- Functional tests
- Edge case tests
- Integration tests

## Tools
- Python HTTP server for open port simulation
- Docker containers for service simulation (optional)
